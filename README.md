# 🤖 MyBot - بوت إدارة مجموعات تيليغرام

بوت متكامل لإدارة مجموعات تيليغرام مبني بـ Pyrogram وRedis.

---

## 📋 المتطلبات

```
Python 3.10+
Redis (محلي أو Upstash/Railway)
```

## ⚙️ متغيرات البيئة

```env
API_ID=12345678
API_HASH=abcdef1234567890abcdef1234567890
BOT_TOKEN=123456789:AABBCCDDEEFFaabbccddeeff
REDIS_URL=redis://localhost:6379          # أو رابط Redis الخارجي
DEV_ID=123456789                          # أيدي المطور (أنت)
```

## 🚀 التشغيل

```bash
pip install -r requirements.txt
python main.py
```

## 📂 هيكل المشروع

```
MyBot/
├── main.py              # نقطة الدخول
├── config.py            # الإعدادات والاتصال بـ Redis
├── requirements.txt
├── Procfile             # للنشر على Heroku/Railway
├── helpers/
│   ├── ranks.py         # نظام الرتب والصلاحيات
│   └── utils.py         # وظائف مشتركة
└── Plugins/
    ├── admin.py          # /start، تفعيل، تعطيل، أقفال
    ├── mute_gban.py      # كتم وحظر (محلي وعام)
    ├── ranks.py          # إدارة الرتب
    ├── welcome.py        # الترحيب والقوانين
    ├── custom_filters.py # فلاتر مخصصة بالكلمات
    ├── custom_commands.py# أوامر مخصصة (أسماء بديلة)
    ├── global_filters.py # فلاتر عامة (لكل المجموعات)
    ├── word_filter.py    # فلتر الكلمات السيئة
    ├── whisper.py        # الهمسة (inline)
    └── group_events.py   # أحداث المجموعة
```

## 📖 دليل الأوامر

### الإعداد الأولي
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `تفعيل` | مالك+ | تفعيل البوت في المجموعة |
| `تعطيل` | مالك+ | تعطيل البوت |

### نظام الرتب
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `رتبة` | الجميع | عرض رتبتك أو رتبة من تردّ عليه |
| `ادمن @user` | مدير+ | تعيين ادمن |
| `مدير @user` | مالك+ | تعيين مدير |
| `مالك @user` | مالك أساسي+ | تعيين مالك |
| `شيل ادمن @user` | مدير+ | إزالة ادمن |
| `قائمة الرتب` | ادمن+ | عرض جميع الرتب |
| `تغيير اسم رتبة ادمن [اسم]` | مالك أساسي+ | تخصيص اسم الرتبة |

### الكتم والحظر
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `كتم` (ردّ) | مدير+ | كتم عضو |
| `كتم @user` | ادمن+ | كتم بالذكر |
| `كتم عام` (ردّ) | مطوّر | كتم في كل المجموعات |
| `الغاء الكتم` (ردّ) | ادمن+ | رفع الكتم |
| `حظر عام` (ردّ) | مطوّر | حظر عام (يُطرد من كل مجموعة) |
| `الغاء الحظر العام` | مطوّر | رفع الحظر العام |

### الفلاتر
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `اضف فلتر [كلمة]` | مدير+ | إضافة رد تلقائي على كلمة |
| `حذف فلتر [كلمة]` | مدير+ | حذف فلتر |
| `الفلاتر` | الجميع | عرض قائمة الفلاتر |

### فلتر الكلمات السيئة
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `اضف كلمة [كلمة]` | مدير+ | حظر كلمة |
| `حذف كلمة [كلمة]` | مدير+ | إزالة حظر كلمة |
| `الكلمات المحظورة` | مدير+ | عرض القائمة |
| `تفعيل فلتر الكلمات` | مدير+ | تشغيل الفلتر |
| `تعطيل فلتر الكلمات` | مدير+ | إيقاف الفلتر |

### الترحيب
| الأمر | الصلاحية | الوصف |
|-------|----------|-------|
| `وضع الترحيب` | مدير+ | تعيين رسالة ترحيب |
| `مسح الترحيب` | مدير+ | حذف الترحيب المخصص |
| `وضع قوانين` | مدير+ | تعيين قوانين |
| `القوانين` | الجميع | عرض قوانين المجموعة |

### الهمسة (Inline)
```
@البوت همستك @username
@البوت مفاجأة @all
```

### الأوامر المخصصة
| الأمر | الوصف |
|-------|-------|
| `اضف امر` | تعيين اسم بديل لأمر موجود |
| `حذف امر [اسم]` | حذف الاسم البديل |
| `الاوامر المضافة` | عرض القائمة |

## ⚠️ ملاحظات

1. يجب رفع البوت **مشرفاً** في المجموعة مع صلاحيات حذف الرسائل.
2. عند أول تشغيل، عيّن أيدي المطور في `DEV_ID`.
3. لتعيين مالك البوت: `r.set(f"{DEV_ID}:owner", "OWNER_ID")` في Redis.
